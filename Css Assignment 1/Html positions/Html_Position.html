Position Property

Parent1: Static Position.

An element with position: static; is not positioned in any special way; it is always positioned according to the normal flow of the page:

Child1: Absolute Position
Parent2: Relative Position.

An element with position: relative; is positioned relative to its normal position.

Child2: Absolute Position
Parent3: Absolute Position.

The position: absolute; property allows you to position element relative to the nearest positioned ancestor.

Child3: Absolute Position
Parent4: Fixed Position.

A fixed-position layout lets the product stay in one place while workers and machinery move to it as needed.

Child4: Absolute Position
